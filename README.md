Minun's Hyprland Dot file
=======

- WM: hyprland

- Terminal: kitty

- Fetch: catnip

- Bar: waybar

- File Manager: Thunar

- Editor: vim

- Browser: Firefox

- Launcher: rofi

Requirements / Gerekli Paketler
============

Yay:

    yay -S tofi hyprpicker rofi-power-menu waybar 

Pacman:

    sudo pacman -S thunar firefox steam kitty neofetch
